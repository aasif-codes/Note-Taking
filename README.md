# #Note

#Note is a simple and clean note-taking web application built using HTML, CSS, and JavaScript.  
It is designed to practice core front-end development concepts and basic DOM manipulation.
## Live Demo
[View Live App](https://aasif-codes.github.io/Note-Taking-App/)

## Features
- Create and delete notes
- Minimal and user-friendly interface
- Runs entirely in the browser

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project is part of my front-end learning journey and focuses on building small, functional applications.
